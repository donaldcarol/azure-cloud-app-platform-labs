\# Azure Cloud App Platform Labs



This repository contains a series of hands-on labs exploring modern Azure application hosting and deployment patterns.



The goal is to cover common cloud engineer tasks such as:



• Deploying web applications using Azure App Service  

• Hosting front-end applications with Azure Static Web Apps  

• Building and deploying containerized applications with Docker and Azure Container Apps  

• Managing container images with Azure Container Registry  

• Automating infrastructure using Terraform  

• Working with DNS, custom domains, and SSL certificates  



\## Lab Structure



Day 1 – Azure App Service  

Deploying a web application, configuring application settings, custom domains and SSL.



Day 2 – Azure Static Web Apps  

Deploying static sites integrated with GitHub workflows.



Day 3 – Containers in Azure  

Building Docker images, pushing to Azure Container Registry, deploying with Azure Container Apps.



Day 4 – Infrastructure as Code  

Using Terraform to provision Azure resources and manage DNS and certificate automation concepts.
